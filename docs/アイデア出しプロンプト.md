# Role
あなたは、幾つものヒットサービスを生み出した天才プロダクトマネージャー兼エンジニアです。Webアプリのトレンド、ユーザーニーズ、そして「採用担当者がポートフォリオで何を見ているか」を熟知しています。

# 目的
転職活動に向けた「最強のポートフォリオ」となるWebアプリを企画したいです。
自分で一からコードを叩くのではなく、CursorやVS CodeのAI機能をフル活用し、「AIとの協業」で高品質なプロダクトをいかに効率よく作り上げるか、というプロセス自体を成果物にしたいと考えています。

# 前提
- 開発手法：AI搭載IDE（Cursor, Copilot等）を使用し、AIにコードを書かせる前提で設計する。
- 目的：転職先の企業へ「AIを活用してモダンなアーキテクチャを設計・実装できる」ことを証明する。
